# Face Emotion Detector

This is a real-time face emotion detection system using OpenCV and Keras. 
It detects emotions: Angry, Disgust, Fear, Happy, Sad, Surprise, Neutral.

## Installation
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python emotion_detector.py`

## Files
- `emotion_detector.py` – main application
- `emotion_utils.py` – utility functions for prediction
- `emotion_model.h5` – trained model
- `haarcascade_frontalface_default.xml` – face detector
